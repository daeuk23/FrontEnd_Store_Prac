Reflection

First of all, I would like to briefly explain Responsive Design & Relevance to web technology, And I will explain the building web for devices into research by dividing to each items, and at the end, I will summarize the facts learned through the course.

  • <What is Responsive Design>

    Responsive design is a type of design technology that uses design technology to automatically adjust the screen size and layout to the type of device. Simply put, the screen size varies depending on the type of device, so it is a technology that sets and displays each screen at a different size (size appropriate for the device). The layout and images are flexibly adjusted depending on the situation, eliminating disadvantages such as space left over, and have the advantage of maintaining consistency regardless of device. In general, it is said that it is designed based on “mobile” screens with small screens and sets how it will change depending on the larger screen.

  • <Relevance to web technology>

    Responsive design impacts many areas in the web development world. In addition to simply setting different display sizes depending on the situation through max-width in CSS that sets the display. Even in the js framework, detailed items such as <img> and <video> can be adjusted, and various display elements such as layout must be directly specified during web development. Therefore, the “use” of responsive design is considered the domain of web technologies. It is also an essential element for consistency across devices.

  • <building web for different devices>

    Based on the above-mentioned Responsive Design, it is prepared to be displayed correctly in various media, and the size can be adjusted by setting CSS queries, layout, and maximum and minimum changes. Furthermore, if necessary, there is a way to optimize it for other devices. For example, conditional expressions and queries can be added so that it can operate accordingly on other OSs, such as mobile interface and touch. Using a virtual testing tool, you can see how it looks. This can resolve any discrepancies that may arise due to use on different OSs. Additionally, there may be a problem with the resolution being broken due to viewing on a larger screen, which can be prevented by setting keywords such as background size and meta. This can solve the display problem.
